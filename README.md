# Amazon Clone (React + Vite)
A React-based Amazon clone project built with **Vite** for fast development and modern bundling. This project simulates an e-commerce website with product listing, search functionality, and a shopping cart.

## About
It includes the following functionalities:
- Product listing fetched from a fake API (`https://dummyjson.com/products`)
- Search products
- Add products to the cart
- Update product quantity in the cart
- Remove products from the cart
- Navigation using React Router (`About`, `Contact`, `Cart` pages)

This project is mainly for **learning purposes** to understand React, state management, routing, and component-based design.

## Features
- Fully responsive product CSS grid layout
- Dynamic product search
- Shopping cart with quantity management
- Navigation bar with links to About, Contact, and Cart pages
- Loader while fetching data
- Error handling for API requests

## Hosted on Vercel:
[https://amazon-clone-react-using-grid.vercel.app/](https://amazon-clone-react-using-grid.vercel.app/)

## Usage
- Use the search bar to filter products by name.
- Click Add to Cart to add a product.
- Go to Cart to increase/decrease quantity or remove items.
- Navigate to About and Contact pages using the navbar.

## Installation
1. Clone the repository:
```bash
git clone https://github.com/poornimarajarao/amazon-clone-react-using-grid.git
```
2. Navigate to the project directory:
```bash
cd amazon-clone-react-using-grid
```
3. Install dependencies:
```bash
npm install
```
4. Start the development server:
```bash
npm run dev
```
5. Open your browser and go to:
```
http://localhost:5173
```
